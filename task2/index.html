Selector # CSS Selector Explanation Purpose HTML element(s) selected
1 .tabComponent The class selector .tabComponent selects any element with class="tabComponent". To apply style rules to elements with class .tabComponent The outmost div element with class="tabComponent", which works as a tab container.
2 div.tabComponent input The type selector div is combined with the class selector .tabComponent to select the outermost div with class="tabComponent". This composite selector is then combined with another type selector input to select all the input type elements. To apply style rules to the radio button input type elements Elements selected by this rule are the radio buttons with id="homeDecor", id="furniture" and id="kitchen".
3 div.tabComponent label The composite selector div.tabComponent label selects all labels. To apply style rules to all label elements Elements selected by this rule are labels.
4 div.tabComponent label:hover The pseudo-class selector :hover works with the div.tabComponent label selector to find the hovered label. To apply style rules to hovered label element Elements selected by this rule are only hovered labels.
5 .tab-content The class selector .tab-content matches all elements with class .tab-content. To apply style rules to elements with class .tab-content Elements selected by this rule are the divs placed right after each label element and with class="tab-content".
6 div.tabComponent input:checked + label The composite selector div.tab-content matches the outmost div element. The pseudo-class selector :checked combined with type selector input finds the checked radio button element. The following combinator + working with type selector label selects the label element, which is the adjacent sibling of the checked input element. To apply style rules to the label element which is the adjacent sibling of a checked input element The element selected by this rule is the label of a check radio button with id="homeDecor", id="furniture" or id="kitchen".
7 div.tabComponent input:nth-of-type(n):checked ~ .tab-content:nth-of-type(n) The pseudo-class :nth-of-type(n) selects the nth given element of its type. The composite selector input:nth-of-type(n):checked matches the nth input of its type if it's checked. The following combinator ~ selects the specified following siblings of the input element. .tab-content:nth-of-type(n) matches the nth element with class .tab-content of its type. To apply style rules to the nth element with class .tab-content when nth input is checked Element selected by this rule is specified by n.
